# FIND-THERAPIST-GHANA API

---
## Stack
*Language : Javascript(nodejs)*
*Database: Mongodb*

---

## Basic Features

- Register a new account(JWT & social authentication)
- Login to account (JWT & social authentication)
- Reset password
- Forgot password
- Create profile
- Edit profile
- Get profile
- Search profile
- Delete account


## User
| field  |  data_type | constraints  |
|---|---|---|
|  firstName  |  string |  required |
|  lastName     | string  | required | 
|  email | string  |  required, unique|
|  password |   string |  required  |
|  confirm_password |   string |  required  |
|  telephoneNumber  |  string |  required |
|  workNumber  |  string |  required |
|  licensingBoard  |  string |  required |
|  licenseNumbber  |  string |  required |

### Forgot-password
- Route: auth/forgot-password
- Method: PATCH
- Body: 
    ```
        {
            "email": "test@user.com"
        }
    ```

- Response: 
    ```
        {
            "status": "Error",
            "message": "Reset link sent to email"
        }
    ```
---

### Reset password
- Route: /auth/resetpassword/{token}
- Method: PATCH
- Body: 
    ```
        {
            "password": "newPassword"
        }
    ```

- Response: 
    ```
        {
            "status": "success",
            "message": "Password reset successfully"
        }
    ```
